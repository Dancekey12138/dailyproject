# Daily Project - GitHub Trending 每日精选

自动采集 GitHub Trending 热门项目，专注于 AI 和工具类项目。

## 📁 目录结构

```
.
├── OriginInfo/          # 任务1: AI+工具项目详细汇总
│   └── YYYY/MM/DD.md
├── AiTrentInfo/         # 任务2: 极客情报内参
│   └── YYYY/MM/DD.md
├── assets/              # 图片资源
│   └── images/
├── INDEX_BY_DATE.md     # 按时间索引
└── INDEX_BY_CATEGORY.md # 按分类索引
```

## 📊 内容说明

### OriginInfo - 项目详细汇总
- 每日精选 10 个 AI/工具类项目
- 每个项目约 600 字深度分析
- 包含基本用法、代码示例、原理说明
- 面向小白，通俗易懂

### AiTrentInfo - 极客情报内参
- 深度分析当日最具冲击力项目
- 热度黑马项目推荐
- 技术风向标趋势分析
- 可执行的行动建议

## 🔍 索引系统

- **时间索引**: 按日期查看所有采集内容
- **分类索引**: 按项目类别快速定位
- **编号映射**: 分类编号 ↔ 时间文件双向映射

## ⏰ 更新频率

每日 23:00 (UTC+8) 自动采集更新

## 🤖 自动化

本仓库由 Hermes Bot 自动维护
- 采集: GitHub Trending API
- 分析: AI 智能总结
- 推送: 微信 + QQ 同步通知
